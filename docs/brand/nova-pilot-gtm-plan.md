## 1. Ideal Pilot District Profile

**Primary target: 3,000–15,000 students (mid-size districts)**
- Have a dedicated DAEP campus or shared DAEP program (40–150 placements/year)
- Director of Student Services has budget authority under $10,000 without board approval
- Lack a dedicated compliance staff member — one person doing the job of three

**Secondary target: 15,000–40,000 students** — longer cycle, use mid-size win as reference first

**Do not pursue in Pilot Phase 1:** Under 1,500 students — volume too low, admin is already in all rooms

### Pain Points Right Now (March 2026)

- **SPED compliance exposure** — missed manifestation determination = OCR complaint risk
- **Manual 30/60/90-day transition plan tracking** — spreadsheets or nothing
- **PEIMS discipline coding** — multi-day extraction process before submission window
- **Repeat offender blind spots** — no real-time alerting across placements
- **Laserfiche dead-end** — daily Excel printout still manually keyed in

### Buyer Map

| Role | Function | Their Fear |
|------|----------|------------|
| **Economic Buyer** | Superintendent or Asst. Supt. for Student Services | OCR complaint, TEA letter, board embarrassment |

| **End User** | Principals, APs, CBCs, SPED Coordinators | Too many paper forms, no visibility into next steps |

## 7. Objection Handling

**"We already use [spreadsheets/STOPit/Frontline]"**
STOPit is a safety reporting tool, not a DAEP compliance workflow. Frontline is national — not built for TEC Chapter 37, no SPED blocking, no Laserfiche integration. "We're Texas-native from day one."

**"We don't have budget this year"**
"When does your new cycle start?" → 60-day pilot at $0, budget decision happens in the fall with real data. Frame as compliance insurance: Waypoint starts at $4,500/year. One PEIMS correction costs more.

**"Our district is too small"**
"How many DAEP placements last year?" If 20+: "Twenty placements = twenty compliance gaps. If one involved a SPED student without a manifestation determination, you have exposure." If under 20: acknowledge honestly, revisit when volume grows.

**"FERPA / data security"**
- Runs on Supabase (AWS US-East, SOC 2 Type II)
- Row-level security enforced at database level — no district sees another's data
- District retains ownership of all student data — no data sharing, no resale
- FERPA-compliant DPA available for district counsel review
- Security details: `clearpathedgroup.com/security.html`

**"Can you integrate with our SIS?"**
"Not yet via automated sync — but you can import students via the same CSV export you already run for PEIMS. The data that matters for DAEP compliance — checklists, transition plans, alerts — doesn't live in your SIS. Waypoint provides the compliance layer your SIS was never designed to give you." Skyward API integration is on the roadmap; don't commit to a date.
