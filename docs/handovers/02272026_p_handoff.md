#### New Hooks — `src/hooks/useDaepDashboard.js`

**`useDaepEnrollmentStats()`**
- Fetches active+approved DAEP incidents with student sub-population fields
- Returns: `{ occupied[], reserved[], byGrade: { [grade]: { total, occupied, reserved, sped, is504, ell, homeless, fosterCare, military, gifted } }, byLevel: { middle, high, elementary }, subPopTotals, total, loading }`
- Campus-scoped via `applyCampusScope` + `spedOnly` filter
- `occupied` and `reserved` are arrays (use `.length` for count)

**`useCapacityCount()`**
- Lightweight district-wide occupied/reserved counts (no campus scope — district total)
- Reads `capacity` from `district?.settings?.daep_capacity` (already in AuthContext, no extra query)
- Bug fix applied: loading state now resolves correctly when `districtId` is null (waypoint_admin)
