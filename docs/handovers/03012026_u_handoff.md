### 3. `src/hooks/useNavigator.js` — 5 new hooks + helpers

| Export | Purpose |
|--------|---------|
| `computeRiskScore(referrals, placements, activeSupportCount)` | Pure fn — weighted 0–100 score. Factors: referral recency/freq, OSS/ISS, DAEP escalation, active supports (protective). |
| `riskLevel(score)` | Returns `'high'` (≥70) / `'medium'` (35–69) / `'low'` (<35) |
| `useEscalationRisk()` | District-wide risk: fetches 90-day referrals + placements + active supports, groups by student, computes scores, returns sorted array |
| `SKILL_GAP_LABELS` | Exported map: key → human label for all 6 skill gap values |
| `useSkillGapData()` | Aggregates skill_gap counts + unique students per gap + recommended interventions |
| `useInterventionEffectiveness()` | Completed supports with before/after data; computes avg reduction, improvement rate, by-type breakdown |
| `useDisproportionality()` | Referral rate (refs ÷ enrollment) by campus + by grade, 90-day window |
| `useNavigatorPilotSummary(schoolYear)` | Full school-year summary: referrals, students, diverted, escalated, ISS/OSS, days removed, avg reduction, top skill gaps |

### 4. 5 New Navigator Pages

| Page | Route | Role Gate |
|------|-------|-----------|
| `NavigatorEscalationPage` | `/navigator/escalation` | staff |
| `NavigatorSkillMapPage` | `/navigator/skill-map` | staff |
| `NavigatorEffectivenessPage` | `/navigator/effectiveness` | staff |
| `NavigatorDisproportionalityPage` | `/navigator/disproportionality` | staff |
| `NavigatorPilotPage` | `/navigator/pilot` | admin/principal |

**NavigatorEscalationPage:** Risk score dashboard. 3 stat cards (High/Medium/Low). Filter tabs. Table: student, grade, campus, score bar (0–100), level badge, trigger chips ("Referral ≤14d", "OSS ≤30d", etc.), active supports count, link to student detail.

**NavigatorSkillMapPage:** Bar chart (skill gap frequency, clickable to highlight row). Table: skill gap label, referral count, unique students, behavior description, recommended interventions (color-coded chips). Empty state: prompts to tag skill_gap on referrals. Note about migration 050.

**NavigatorEffectivenessPage:** 4 metric cards (tracked, avg reduction, improved count, improvement %). Grouped bar chart (Before vs After by support type). Detail table: student, support type, completed date, before/after counts, delta arrow, outcome notes. Empty state: prompts to fill incidents_before/after on completed supports.

**NavigatorDisproportionalityPage:** Horizontal bar chart (campus referral rate, color-scaled low→high). Campus table with vs-avg column. Vertical bar chart (by grade). Color legend.

**NavigatorPilotPage:** School year selector. 4 large headline metrics (total referrals, students served, diverted, escalated). Secondary row (ISS, OSS, days removed, diversion rate). Supports row (active, completed, avg reduction). Top skill gaps bar chart. Auto-generated leadership narrative paragraph.
