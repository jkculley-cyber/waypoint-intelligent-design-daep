## Lessons This Extended Session

- **The homepage was T-TESS-centric in places the code never was.** The Apex app has supported multiple rubrics including IB since migration 009. The IB market is a legitimate TAM (independent schools, international networks) and leaving them out of marketing copy was sloppy.
- **Skip links are trivial to implement but often skipped.** Injection pattern is straightforward: one CSS block, one anchor tag after `<body>`, one `id="main"` on the hero. Done 16 times in ~20 minutes.
- **"Hero tag" style badges work better than emoji for brand signaling.** Emojis render inconsistently across OSes (Apple vs Windows vs Android) and can look juvenile on a procurement page. Colored letter badges in the brand font are platform-consistent.
