## Lessons worth keeping
- **Ground audits in current code, not the last audit's snapshot.** The parallel-dossier pass surfaced a shipping race-label bug and a manifestation-MDR hole that prior R3 verdicts (84%) missed because they predated migration 070 / never exercised the save gate.
- **Check the right CI signal.** Beacon's *Pages* deploy (the real one) passed on the exact commit while a secondary *Workers Builds* check failed — don't merge over a red check just because another check is green; don't panic over a red check that isn't the deploy path either. Read the log.
- **A "quick guard" that breaks a sibling product isn't quick.** N3's `UNIQUE(district_id)` looked trivial until you remember Waypoint shares the `campuses` table and is legitimately multi-campus.
- **Verify "applied" with a query, every time.** Migration 084 got a real `pg_constraint` confirmation, not a "done." Local key being the disabled legacy one blocked the automated check — surfaced as a side finding.
