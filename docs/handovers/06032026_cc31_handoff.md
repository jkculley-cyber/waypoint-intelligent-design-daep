## Lessons worth keeping
- **Reconnecting a Cloudflare Pages git integration re-runs framework auto-detection.** For a static site living in a monorepo subdir, that silently sets a Vite build command and deploys the *app* over the site. After any reconnect, immediately re-assert Framework=None + empty build command before deploying.
- **Validate infra changes under observation, then trust them.** The controlled retry (watched) converted "probably safe" config into "proven safe" without waiting for an unattended push to surprise-break production.
- **Cloudflare Rollback is the fast undo** — know where it is *before* you need it. Today it restored the marketing site in under a minute.
- **An audit agent scoped to one repo can report a false "missing".** B1's "no endpoint" was wrong — the code was in a sibling repo. When a finding says "X doesn't exist," confirm the search covered the right repo/deploy source.
