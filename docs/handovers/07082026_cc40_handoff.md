## 4. AI unit-economics — the "why did earlier Apex pilots drain?" answer

**Per-observation AI is cheap: ~$0.03–0.08.** Groq `whisper-large-v3-turbo` transcription ~1–3¢ + Sonnet 4.6 draft (max 2048 out) ~2–5¢. A 150-obs/year coordinator ≈ $5–12/yr. The current Groq+Sonnet stack is already ~5–10× cheaper than the OpenAI-Whisper stack the earlier versions used.

**The actual drain: `002_morning_brief_cron.sql`.** A daily `pg_cron` job calls `generate-morning-brief` (Sonnet 4.6, max 1200 out) for **every** principal with `onboarding_complete AND morning_brief_enabled` — **no subscription check**. So every trial/lapsed signup gets ~$5–9/yr of daily AI forever. Cost scales with **signups**, not customers → the classic "drained in pilots, no purchases" pattern, still live.

**The fix the user is converging on — tier it:**
| | Essentials (no voice) | Pro / AI (paid, per-campus) |
|---|---|---|
| AI cost to serve | **~$0** (pure DB workflow) | **~$16–26/campus/yr** |
| Content | typed obs, self-study, PD, policies, docs, checklist | + voice→transcription→AI drafts + AI brief + dashboard summary |

At a $499/campus price the Pro AI COGS is ~4–6% → **~95% gross margin**, and free/Essentials signups cost ~$0. Trial gets a small **capped AI taster** (e.g. 5 free drafts, one-time ~$0.30) to demo the "wow", then paywall. **Gate AI in the edge functions** (transcribe-observation, ib-coaching-draft, ib-health-summary, generate-morning-brief) + the cron — not just the UI (functions are directly callable, same class as the RLS bug). Blocker: there is no real IB tier field yet — `ib_school` boolean can't distinguish Essentials from Pro.
