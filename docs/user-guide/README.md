## Key Compliance Facts

Waypoint enforces **Texas Education Code Chapter 37** and **IDEA (20 U.S.C. § 1400)** requirements:

- **Hard block:** SPED and 504 students cannot be placed in DAEP until every required compliance step is complete and documented.
- **10-day rule:** SPED students with cumulative removal reaching 10+ school days trigger automatic IDEA protections. Waypoint blocks submission and requires MDR before proceeding.
- **30/60/90-day reviews:** Transition plans have scheduled review windows. Overdue reviews are prominently flagged.
- **Manifestation Determination:** Required for every SPED/504 DAEP placement. System requires both the determination AND the result (IS or NOT a manifestation) to be recorded.
- **Audit trail:** Every action in Waypoint is timestamped and tied to a user account. This is your evidence in a TEA audit.
