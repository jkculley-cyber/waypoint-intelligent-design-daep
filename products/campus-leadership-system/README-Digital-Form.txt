FILE: Investigation-Intake-Form.html

HOW TO OPEN:
  Double-click the file OR
  Right-click → Open with → Chrome / Edge / Safari / Firefox
  
  Works on: Windows, Mac, Chromebook, iPad
  Does NOT require: internet connection, account, or login
  Works completely offline after download.

BASIC WORKFLOW:
  1. Open the file in your browser
  2. Select the offense type (top card grid)
  3. Fill out the 10 sections
  4. Click "Export to Toolkit" to download a CSV row

CONNECTING TO THE ADMIN COMMAND TOOLKIT:
  After clicking "Export to Toolkit":
  1. Open Admin-Command-Toolkit-Texas-Edition.xlsx
  2. Go to the "Incident Intake" tab
  3. Click the first empty row in column A
  4. Paste (Ctrl+V or Cmd+V)
  All 14 connected tabs update automatically.

SAVING A DRAFT:
  Click "Save Draft" to store in browser memory.
  Click "Load Draft" to retrieve it.
  Note: drafts are stored in your browser — clearing 
  browser data will erase them. Use "Export Full Record"
  for permanent archiving.

PRINTING:
  Click "Print Record" or use Ctrl+P / Cmd+P.
  The sidebar and buttons are hidden in print view.
  Select "Save as PDF" in the print dialog.

BROWSER COMPATIBILITY:
  Best: Google Chrome or Microsoft Edge
  Good: Safari, Firefox
  Not supported: Internet Explorer
