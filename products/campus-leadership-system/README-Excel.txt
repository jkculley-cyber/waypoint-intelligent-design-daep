OPENING IN MICROSOFT EXCEL:
  Double-click the .xlsx file. Enable editing if prompted.
  Enable macros if prompted (no macros in these files —
  safe to enable or dismiss).

OPENING IN GOOGLE SHEETS:
  1. Go to drive.google.com
  2. Click New → File Upload → select the .xlsx file
  3. Once uploaded, right-click the file
  4. Select Open with → Google Sheets
  All formulas, dropdowns, and conditional formatting transfer.

FIRST STEPS:
  1. Open the Cover tab — fill in District Name, Campus, School Year
  2. Review the instructions on the Cover tab
  3. Start entering data in the first data tab (Roster or Intake)
  4. Auto-calculated columns (marked ⚙) fill in automatically —
     do not type in those cells

COLOR CODING:
  Purple headers = section labels (read only)
  White cells = your input
  Gray cells = auto-calculated (do not edit)
  Red rows = overdue / requires immediate action
  Gold rows = approaching deadline (warning)
  Green rows = complete / compliant

CONDITIONAL FORMATTING:
  Colors apply automatically based on the data you enter.
  No setup required.

DATA VALIDATION DROPDOWNS:
  Many columns have dropdown menus.
  Click the cell → click the dropdown arrow to see options.
  All dropdown values come from the Lists tab.
