USING THE WORD (.docx) VERSION:
  • Open in Microsoft Word or Google Docs
  • Click any underlined field and type your information
  • Check boxes by clicking them (Word) or typing an X next to them (Docs)
  • Save a copy with the Case ID in the filename for filing
  • Print or Save as PDF when complete

NEED A PDF?
  Open the .docx and use File → Print → Save as PDF,
  or print and complete by hand.

FILING:
  • Original: Student's locked disciplinary file
  • Copy: Central office (if mandatory removal)
  • Copy: SPED file (if student has IEP or 504)

IMPORTANT:
  These templates are CONFIDENTIAL FERPA education records.
  Do not share with unauthorized parties.
  Retain per your district's records retention policy.

PAIRS WITH:
  The Digital Investigation Intake Form (.html file, separate listing)
  lets you fill out the investigation digitally and export the data
  directly to the Administrator's Command Toolkit — no double entry.
